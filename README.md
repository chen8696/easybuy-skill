# easy-buy · 轻松购

> AI 购物决策助手 — 让你只买对的，不花冤枉钱

## 它解决什么问题？

| 痛点 | 轻松购怎么解决 |
|------|---------------|
| 看100篇测评还是不知道买哪个 | 一句话说需求，AI 自动搜集+筛选+验证 |
| KOL 推荐全是软文，不敢信 | Tier 分级 + 软文识别 checklist，只信真评测 |
| 同款商品不同平台差价几百 | 5 大平台实时比价 + 历史价格验证 |
| 优惠券藏得深，错过省几百 | 政府补贴 + 消费券 + 平台会员 + 隐藏优惠全聚合 |
| 只推贵的，预算外强推 | 严格在预算内筛选，不好就直说"不建议买" |
| 推了10个选择，更懵了 | 最多 3 个最优推荐，含对比表格 + 购买链接 |

## 核心能力

1. **信息源甄别与交叉验证** — Tier 1/2/3 分级，软文识别，多源共识才纳入候选
2. **跨平台比价** — 京东/天猫/拼多多/抖音电商/品牌官网，至少 3 渠道交叉验证
3. **隐藏优惠聚合** — 政府补贴、地方消费券、平台会员价、品类券、支付优惠、返利
4. **溢价识别与白牌平替** — 拆解成本结构，找同厂白牌/二线品牌/上代旗舰
5. **安全红线** — 食品安全审核、反水军检测、搜索无果兜底，该不买就直说
6. **最多 3 推 + 对比表** — 每个推荐含核心优势 + 真实缺点 + 优惠后到手价 + 购买链接

## 快速开始

### 安装

将本项目复制到你的 skills 目录：

```bash
# 用户级（所有项目通用）
cp -r easy-buy ~/.workbuddy/skills/

# 或项目级（团队共享）
cp -r easy-buy .workbuddy/skills/
```

### 使用

在 WorkBuddy 对话中直接说：

- "帮我推荐 2000 以内的扫地机器人，给爸妈用，家里有宠物"
- "下个月去成都旅游 4 天，预算 300-500 住春熙路附近"
- "通勤耳机 1000 左右，入耳还是头戴好？"

## 工作流程

```
需求澄清 → 多维度搜索 → 信息源甄别+交叉验证 → 成本分析+溢价剔除 → 跨平台比价+优惠聚合 → 生成推荐
   ↑           ↑                ↑                      ↑                      ↑
 🔴CHECKPOINT  6种搜索策略    Tier分级+软文识别     成本拆解+白牌平替      5平台比价+隐藏优惠
```

## 安全设计

- **15 条禁止清单**：覆盖推荐/操作/流程三类红线（见 SKILL.md Anti-patterns & Blacklist）
- **2 个 CHECKPOINT**：需求确认 + 候选确认，不跳过用户确认
- **3 个 failover 分支**：信息源降级 / 候选池空 / 历史价格缺
- **2 个 STOP 安全红线**：食品安全 + 候选安全门
